## ОСНОВНЫЕ ХАРАКТЕРИСТИКИ

| Параметр | Описание |
|----------|---------|
| **Скорость выполнения** | Примерно как улитка, которой дали кофе.  |
| **Скорость разработки** | Пишется быстрее, чем вы успеваете пожалеть о выборе языка. Потом поддерживается... ну, допустим, тоже быстро. Быстро надоедает. |
| **Типизация** | Динамическая. Настоящие джедаи узнают тип переменной только в момент её падения в продакшене. `mypy` — это просто декорация. |
