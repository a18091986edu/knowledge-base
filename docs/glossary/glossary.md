<a id="glossary-linter"></a>
**Линтер** - инструмент для анализа исхоного кода с целью выявления и исправления ошибок, уязвимостей, а также нарушений стандартов стиля кодирования

<a id="glossary-formatter"></a>
**Форматтер** - инструмент, который автоматически применяет единые правила форматирования pep8 к исходному коду Python, делая его единообразным и более читаемым
